# YOU Eyewear Website

Modern gözlük koleksiyonu web sitesi.

## GitHub Pages Deployment

Bu proje GitHub Pages'te otomatik olarak deploy edilir.

### Manuel Deploy

```bash
npm run deploy
```

### Local Development

```bash
npm run dev
```

## Özellikler

- React + Vite
- Tailwind CSS
- Responsive tasarım
- SPA routing
- GitHub Pages uyumlu
